# agenda
Aenda webApp creada con Java PrimeaFaces y Hibertnate
Esta aplicacion es una aplicacion que es capaz de crear tareas ponerlas en el calendario arignar una prioridad creada por el usuario y oraginarla en tipo de tareas que  podrian ser por 
ejemplo laborales caseras de estudios o la que desee crear el usuario, tiene un sistema de registro y logueo  tambien
